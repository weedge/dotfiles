dotfiles
========

My dotfiles.

Install
-------

```bash
$ git clone git@github.com:zachwhaley/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ ./bootstrap
```

Vim plugins
------------------

Install

```bash
$ ./vpm install git@github.com:tpope/vim-sensible.git
```

Update

```bash
$ ./vpm update
```

Remove

```bash
$ ./vpm remove vim.symlink/bundle/vim-sensible
```
